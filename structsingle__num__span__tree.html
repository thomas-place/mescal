Detailed Description

Type used to store the information needed to compute the FOLDING of a R-class.

Stores a spanning tree for the R-class plus the list of dropped edges in the construction.

Remarks
Only partial information is stored for the spanning tre: span_trees[s][a] is the number of occurrences of the letter a on the path from the root of the tree to s (the root being an arbitrary element in the R of s).
The dropped edges are stored in a dequeue. An edge (r,a,s) is stored in the dequeue and represented by the integer r * size_alpha + a (this is a code since the graph is deteministic).
