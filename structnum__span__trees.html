Detailed Description

Type used to store the information needed to compute the AMT-kernel.

AMT-kernels

Stores a spanning tree for each regular R- or L-class (depending on the arguments) plus the list of dropped edges in the construction.

Remarks
Only partial information is stored for each spanning tree: span_trees[s][a] is the number of occurrences of the letter a on the path from the root of the tree to s (the root being an arbitrary fixed idempotent in the R or L class of s).
The dropped edges are stored in dequeues. An edge (r,a,s) is stored in the dequeue of the class of r and represented by the integer r * size_alpha + a (this is a code since the graph is deteministic).
