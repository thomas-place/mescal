Determinization of a NFA with the subset construction.

Remarks
The input Boolean is used to indicate whether the names of the states have to be saved (this only impacts display). The name of a state is the set of states to which it corresponds in the subset construction.
Returns
The complete DFA built with the subset construction.
Parameters
AThe NFA.
